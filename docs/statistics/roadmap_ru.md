# Роадмап: Функция Статистики и Унификация UI (По-русски)

## 1. Расширение Слой Данных (GameManager.gd)
- **Новые переменные**:
  - `total_games_played` (игры всего): int
  - `total_time_played` (время всего): float
  - `total_kills` (убийств всего): int
  - `total_boss_kills` (убийств боссов всего): int
  - `total_deaths` (смертей всего): int
  - `total_shards_collected` (собрано осколков): int
  - `total_damage_dealt`: float (всего урона)
  - `total_distance_traveled`: float (пройденная дистанция)
  - `total_potions_consumed`: int (выпито зелий)
  - `total_dashes_made`: int (сделано рывков)
  - `max_run_time`: float (лучшее время)
  - `max_run_kills`: int (лучший результат убийств)
  - `max_run_level`: int (макс. уровень за забег)
  - `max_kills_per_minute`: int (макс. убийств в минуту - рекорд)

- **Логика**:
  - Обновить `save_game()` / `load_game()` для поддержки новых данных.
  - Реализовать `update_post_run_stats()` для вызова при смерти или победе игрока.

## 2. Унифицированный Скрипт Эффектов (ui_effects.gd)
- Создать глобальный скрипт (Autoload) для объединения внешнего вида.
- **Методы**:
  - `apply_premium_style(control: Control)`: Устанавливает темно-кроваво-золотой стиль для панелей.
  - `setup_button(button: Button)`: Стандартная анимация при наведении и нажатии.
  - `setup_standard_label(label: Label, size: int)`: Единые настройки шрифтов.

## 3. Меню Статистики (statistics_menu.tscn)
- **Верхняя Панель**: Название "СТАТИСТИКА" и кнопка "Назад".
- **Доступ**: Только из Главного меню.
- **Основной Блок**: `TabContainer` с двумя страницами:

  - **Лучшая Игра**: Сетка со всеми рекордами за один лучший забег.
  - **Общая Статистика**: Накопительные данные за все время игры.
- **Визуализация**: Использование нового стиля из шага 2.

## 4. Рефакторинг Главного Меню (main_menu.gd / tscn)

- **Добавление**: Добавить кнопку "СТАТИСТИКА" в основной список кнопок меню.
